下面的仅记录日常工作中经常用到的一些shell命令，希望对大家有所帮助。
后面也会时时更新该文档。

设置http代理

export https_proxy=https://ip:port
export http_proxy = http://ip:port

git设置http代理

git config –global http.proxy “127.0.0.1:8087

查看文件集文件大小

du -h dir
举个例子：
` du -sh xmldb/
du -sm | sort -n //统计当前目录大小 并安大小 排序
du -sk | sort -n
du -sk | grep guojf //看一个人的大小
du -m | cut -d “/“ -f 2 //看第二个/ 字符前的文字
查看此文件夹有多少文件 /// 有多少文件
du xmldb/
du xmldb///* |wc -l

centos 安装 tshark

yum whatprovides tshark

tshark 使用方法

tshark -r tcp_16.pcap -V -Y frame.number==7819
-r 指定源文件
-V 尽可能详细展示包
-Y display filter
-R display filter
tshark -r tcp_16.pcap -V -Y “ip.src==125.39.240.75 && frame.number==27289”

设置linux时间

date –set=”20140125 09:17:00”

vim 中全局替换

%s/src/dest/g

pip 使用国内镜像源下载

pip install -i https://pypi.douban.com/simple pandas

linux创建用户操作

useradd 用户名 -g 组名–G 组名-d Home 目录名-p 密码
userdel 用户名
groupadd newgroupch
groupdel newgoup
passwd 用户名
chmod 【who】[+|-|=][mode] 文件名
chown 【用户：组】 文件

举例:
修改文件夹sten及子文件夹的组为stenographer

chgrp -R stenographer sten

修改文件夹sten及子文件夹的用户为stenographer

chown -R stenographer sten
